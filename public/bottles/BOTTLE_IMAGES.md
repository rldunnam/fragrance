# Bottle Images — Download Guide

Place all downloaded images in: **`public/bottles/`**

## Instructions

1. Create the directory: `mkdir -p public/bottles`
2. For each row below, visit the brand site, find the fragrance product page
3. Right-click the main product bottle image → **Save Image As**
4. Save with the exact filename shown in the **File** column
5. Drop all files into `public/bottles/`

**Tips:**
- Look for a clean bottle shot on a white or transparent background
- Most brand sites have a "press" or "media" section with high-res images
- Aim for portrait-oriented images (taller than wide) — these fit the card banner best
- PNG or JPG both work; name the file exactly as shown (`.jpg` extension)

## Adding More Images Later

Any fragrance in `lib/fragrances/data.ts` can have an image added at any time.
Simply add `imageUrl: '/bottles/{id}.jpg'` to the fragrance entry and
drop the corresponding image file in `public/bottles/`.

## Card Fallback

If a `.jpg` file is missing for an entry that has `imageUrl` set, the Next.js
`<Image>` component will show a broken image icon. To avoid this, only add
`imageUrl` to entries where you have the file ready — or keep the field present
and rely on the `onError` handler in the card component to fall back to the
bottle silhouette icon automatically.
